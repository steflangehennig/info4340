## Assignment

Choose one task and conduct a systematic evaluation of AI-generated outputs.

### Components

1. **Task description and prompt(s)** — what you're evaluating, the full prompts used, model(s) and settings
2. **Three outputs** — generated with different prompts, models, or settings
3. **Rubric** — 4–6 dimensions with 0–4 scoring anchors specific to this task
4. **Score table with justifications** — each output scored on each dimension with 1–2 sentence justifications
5. **Failure mode discussion** — what patterns did you observe (hallucination, omission, overconfidence, etc.)?
6. **Recommendation** — acceptable, acceptable with revision, or unacceptable — with evidence
7. **AI use log**

## Rubric

| Criterion | Excellent (5) | Adequate (3) | Needs revision (1) |
|---|---|---|---|
| Task and prompts | Clear task with documented prompts | Task identified but prompts incomplete | Vague task or missing prompts |
| Rubric quality | Specific, usable scoring anchors | Dimensions present but anchors vague | Missing or no scoring scale |
| Scoring rigor | All scored with specific justifications | Scores present but justifications thin | No justifications |
| Failure analysis | Specific modes identified with examples | Some modes noted | No failure analysis |
| Recommendation | Evidence-based with clear reasoning | Present but weakly supported | Missing or unsupported |
| AI use log | Detailed process documentation | Present but incomplete | Missing |

**Total: 30 points**
